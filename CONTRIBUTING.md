# Contributing to AccreditEx

First off, thank you for considering contributing to AccreditEx! It's people like you that make AccreditEx such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, [make one](https://github.com/ashrafmusa/accreditex/issues/new)!

## Fork & create a branch

If you want to contribute, please fork the repository and create a new branch. This is where you'll be making your changes.

## Get the latest changes

Before you start making changes, please make sure you have the latest version of the `main` branch.

```bash
git checkout main
git pull
```

## Commit your changes

Once you've made your changes, please commit them with a clear and concise commit message.

```bash
git commit -m "feat: add new feature"
```

## Push your changes

Push your changes to your forked repository.

```bash
git push origin my-new-feature
```

## Create a pull request

Once you've pushed your changes, you can create a pull request to the `main` branch of the AccreditEx repository.
